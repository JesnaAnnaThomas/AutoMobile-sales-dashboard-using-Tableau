# AutoMobile-sales-dashboard-using-Tableau
The Automobile Sales Dashboard is an interactive data visualization project developed using Tableau to analyze and present automobile sales performance across different regions, categories, and time periods. Built using a structured dataset containing attributes such as order date, product name, category, region, quantity, sales, and profit, the dashboard transforms raw data into meaningful insights for better decision-making. It includes key features such as KPI indicators for total sales, profit, and quantity, a sales trend line chart to identify patterns and peak periods, bar charts for category-wise performance comparison, a map visualization for regional analysis, and sections highlighting top customers and best-selling products. Interactive filters are also integrated to allow dynamic exploration based on region, category, and time. The project aims to identify sales trends, evaluate regional and product performance, and support strategic business decisions, ultimately simplifying complex data into a clear and intuitive visual format for stakeholders.
